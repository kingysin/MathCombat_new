# Math_Combat_Game


## Установка Qt Visual Studio Tools

- Открыть окно управления расширениями ```Extensions/ManageExtensions```
- Ввести в поле поиска 'qt'
- Выбрать в списке 'Qt Visual Studio Tools'
- Нажать на кнопку 'Download'
- Переоткрыть IDE

## Установка библиотеки Qt 6.4.1

- Распаковать архив ```Tools/Qt``` на системный диск ```C```
- Убедиться в корректности пути ```C:\Qt\6.4.1\msvc2019_64```
- Запустить IDE в любом проекте или без него ('Continue without code')
- Открыть настройки версиями Qt ```Extensions/Qt VS Tools/Qt Versions```
- Добавить новую версию, указав наименование '6.4.1' и путь 'C:\Qt\6.4.1\msvc2019_64'
